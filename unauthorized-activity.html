✅ Unauthorized Activity – Checklist

🔔 Trigger

  • Customer reports a loan was opened without their permission
  • e.g., identity theft, fraud, family/friend misuse

🧭 Ask Probing Questions

  • Could a family member or friend have opened this?
  • Have you filed a police report?
  • Have you experienced identity theft before?
  • Would you like to file an Unauthorized Activity claim with Affirm?

📧 Contact Info

  • Ask for email or mailing address to follow up
  • If customer refuses, inform them they can submit info at Affirm Help Center

☎️ TCN Actions

  • Stop calls for 1 month while the investigation is in progress

🛠️ Agent Portal

  • Go to: Agent Portal > Manage Person > Account Tools > Report Unauthorized Activity
  • Select all loans the customer disputes
  • Turn off AutoPay on each loan

📂 Salesforce Case Creation

  • Create a case type: Unauthorized Activity
  • Fill out Dynamic Fields:
    • Contact Method: Phone
    • Contact First & Last Name
    • Active Loans: Yes/No
    • Disputed Product ID(s): All relevant Loan IDs
    • Unauthorized Activity Type
    • Tool Updated: Yes/No (depending on whether step was done in AP)
    • Sufficient Documentation to Proceed: No
    • Status: In Progress
    • Additional Comments: Summarize conversation + Loan IDs

🔁 Routing

  • Reassign case to Next Team
  • Do NOT create child cases for associate accounts

🛠️ Mordor Actions

  • Go to: Mordor > Users > Adjust User Collection
  • Enter User ARI
  • Under 'User Collection Status', select reason: Other
  • Comment: "FO Intake – Case #[case number]"
  • Click Update User Collection
  • Repeat for each affected account

📉 Collection Records

  • Create Collection Record(s)
