✅ Harassment – Checklist

🔔 Trigger

  • Customer claims Affirm is harassing them

🧭 Determine Specificity of the Complaint

  • ✅ If the customer mentions a specific channel (calls, emails, SMS):
    • Follow the Update User Communication Preferences checklist
  • ❌ If the customer does NOT mention a specific channel (e.g., "Affirm is harassing me"):
    • Follow the Cease and Desist checklist

💬 Rocket Chat Message

  • Post Message in Rocket Chat:
  • Phone number reflected on TCN: [insert phone number]
    ARI: [insert ARI]
    Discription of the complaint: [insert discription]

🛠️ Agent Portal

  • If following Update Usercomm Preferences: Remove selected communication method(s): Phone / Email / SMS
  • If following Cease and Desist: Apply Cease and Desist Tool

☎️ TCN System (if applicable)

  • Stop calls on TCN: Never
  • TCN Disposition:
  • "Do Not Call" (if comm preferences updated)
    "Cease and Desist" (if general harassment claim)

📉 Collection Records

  • Create Collection Records for all affected accounts
