✅ Wrong Number – Checklist

🔔 Trigger

  • A caller or voicemail indicates that Affirm is contacting the wrong number, or the number is no longer in service.

☎️ TCN Actions

  • Stop calls on TCN: Never
  • TCN Disposition: Inactive Number/Wrong Number

🛠️ Agent Portal Actions

  • Release the phone number from the account

📂 Salesforce Case Creation

  • Create case type: Update Phone Number
  • Close the case immediately
  • Create a child case for each associated account
  • Fill out Additional Comments:
    • "Phone number was released due to a wrong number claim – [Your Name + Initial] / [Date]"

📉 Collection Records

  • Do not create Collection Records for wrong number cases
For any confusion or latest updates, please refer to the official procedure PDF.
